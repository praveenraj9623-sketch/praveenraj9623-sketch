## 🌟 Featured Project — Enterprise Document Intelligence Platform v2.0

> An AI document intelligence platform that analyzes business PDFs using **RAG, ChromaDB, LangChain tool-calling, Groq LLM, MongoDB metadata, and FastAPI-ready architecture**.

[![Live App](https://img.shields.io/badge/Live_App-Open-22c55e?style=for-the-badge&logo=streamlit&logoColor=white)](https://enterprise-document-agentic-ai-v20-fqmmi6vktezorfdwxrrcep.streamlit.app/)
[![GitHub Repo](https://img.shields.io/badge/GitHub-Repository-181717?style=for-the-badge&logo=github&logoColor=white)](https://github.com/praveenraj9623-sketch/enterprise-document-agentic-ai-v2.0)
[![GenAI](https://img.shields.io/badge/GenAI-RAG_Pipeline-2563EB?style=for-the-badge)](https://enterprise-document-agentic-ai-v20-fqmmi6vktezorfdwxrrcep.streamlit.app/)
[![Agentic AI](https://img.shields.io/badge/LangChain-Tool_Calling_Agent-7c3aed?style=for-the-badge)](https://github.com/praveenraj9623-sketch/enterprise-document-agentic-ai-v2.0)

**What makes this stand out:** the app lets users upload business PDFs, extract entities, classify clauses, review risk, compare documents, generate executive summaries, and ask document-specific questions using retrieval-based AI.

```text
PDF Upload
    │
    ▼
Text Extraction + Chunking
    │
    ├──▶ Entity Extraction
    ├──▶ Clause Classification
    ├──▶ Risk Scoring
    ├──▶ ChromaDB / Semantic Retrieval
    └──▶ LangChain + Groq Answer Generation
    │
    ▼
Executive Summary + Document Q&A + Evidence Trace
```
